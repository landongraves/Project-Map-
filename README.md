# Project Map

**A simple, open-source tool for mapping and planning.**

Project Map is a work-in-progress web app that lets you organize ideas, plans, and locations all in one place. You can create projects with an interactive map and a planning area where you can add titles, subtitles, lists, rules, notes, and other information. Projects can be saved and shared as files so others can open them and see the full plan exactly as you created it.

## Features (In Development)
- Interactive web map with markers
- Planning area with titles, subtitles, and lists
- Save projects locally and share via files
- Flexible layout for ideas, rules, notes, and support info

## How You Can Help
Project Map is still early in development. Contributions are welcome, including:
- Improving the map interface
- Enhancing planning tools and layouts
- Adding export/import functionality
- Making the app more user-friendly and collaborative

## Getting Started
1. Clone or download the repository
2. Open `index.html` in a web browser
3. Start creating your project and save/export as needed
